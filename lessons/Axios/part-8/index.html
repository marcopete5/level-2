8. Axios GET Requests

Axiosâ€™s get method retrieves data from an API, automatically parsing JSON into response.data.

  • Syntax: axios.get(url) returns a promise.
  • Response: Access data via response.data.

Example: Displaying Todos

Get and display todos from V School API:


<script src="https://unpkg.com/axios/dist/axios.min.js"></script>
<script>
axios.get("https://api.vschool.io/yourname/todo")
    .then(response => {
        for (let todo of response.data) {
            const h1 = document.createElement("h1");
            h1.textContent = todo.title;
            document.body.appendChild(h1);
        }
    })
    .catch(error => console.log("Error:", error));
</script>
