7. Introduction to Axios

Axios is a JavaScript library for making HTTP requests, simplifying AJAX calls with a clean API. It’s promise-based and widely used in React for its ease of use.

  • Why Axios? Easier syntax than fetch, automatic JSON parsing, robust error handling.
  • Installation:
    • CDN: Add <script src="https://unpkg.com/axios/dist/axios.min.js"></script> to your HTML for browser use.
    • NPM: Run npm init -y, then npm install axios, and use const axios = require("axios") in Node.js.

Example: Axios with SWAPI

Fetch Luke Skywalker’s data using Axios:



<script src="https://unpkg.com/axios/dist/axios.min.js"></script>
<script>
// Make a GET request to SWAPI
axios.get("https://swapi.co/api/people/1")
    .then(response => console.log(response.data.name)) // Logs "Luke Skywalker"
    .catch(error => console.log("Error:", error)); // Logs errors
</script>

Using Config Objects

Axios allows a config object to customize requests, such as adding headers for content preferences or authentication. 


// Fetch data with a config object specifying headers
axios.get("https://swapi.co/api/people/1", {
    headers: {
        "Accept": "application/json", // Request JSON response
        "User-Agent": "My-App/1.0" // Custom identifier for the app
    }
})
    .then(response => console.log(response.data.name)) // Logs "Luke Skywalker"
    .catch(error => console.log("Error:", error)); // Logs errors
