7. Introduction to Axios

Axios is a JavaScript library for making HTTP requests, simplifying AJAX calls with a clean API. It’s promise-based and widely used in React.

  • Why Axios? Easier syntax than fetch, automatic JSON parsing, better error handling.
  • Setup: Use via CDN or npm for Node projects.

Example: Axios with SWAPI

Fetch Luke Skywalker’s data:


<script src="https://unpkg.com/axios/dist/axios.min.js"></script>
<script>
axios.get("https://swapi.co/api/people/1")
    .then(response => console.log(response.data.name)) // "Luke Skywalker"
    .catch(error => console.log("Error:", error));
</script>
