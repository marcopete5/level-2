3. AJAX and Fetch

AJAX (Asynchronous JavaScript and XML) lets web pages update without reloading by fetching data in the background. Fetch is a modern way to perform AJAX requests.

  • AJAX Steps: Event triggers request, server processes, response updates page.
  • Fetch Role: Simplifies AJAX with promises, replacing complex XMLHttpRequest.

Example: AJAX with Fetch

Update a page with SWAPI data:


fetch("https://swapi.co/api/people/1")
    .then(res => res.json())
    .then(data => {
        document.getElementById("name").textContent = data.name;
    })
    .catch(err => console.log("Error:", err));
