9. Axios POST Requests

Axiosâ€™s post method sends data to create resources on a server, like adding a todo to the V School API.

  • Syntax: axios.post(url, data) returns a promise.
  • Use Case: Create new records (e.g., todos).

Example: Posting a Todo

Submit a new todo:


<!-- HTML -->

<form name="todoform">
    <input name="title" placeholder="Title">
    <input name="description" placeholder="Description">
    <button type="submit">Add Todo</button>
</form>
<script src="https://unpkg.com/axios/dist/axios.min.js"></script>




// JavaScript

const todoForm = document.todoform;
todoForm.addEventListener("submit", function(event) {
    event.preventDefault();
    const newTodo = {
        title: todoForm.title.value,
        description: todoForm.description.value
    };
    axios.post("https://api.vschool.io/yourname/todo", newTodo)
        .then(response => console.log(response.data))
        .catch(error => console.log("Error:", error));
});
