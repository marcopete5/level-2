11. Axios DELETE Requests

Axiosâ€™s delete method removes resources from a server, like deleting a todo in the V School API.

  • Syntax: axios.delete(url) targets a resource by ID.
  • Use Case: Remove records (e.g., completed todos).

Example: Deleting a Todo

Delete a todo:


<button id="delete-button">Delete Todo</button>
<script src="https://unpkg.com/axios/dist/axios.min.js"></script>
<script>
const button = document.getElementById("delete-button");
button.addEventListener("click", function() {
    axios.delete("https://api.vschool.io/yourname/todo/your-todo-id")
        .then(response => console.log(response.data))
        .catch(error => console.log("Error:", error));
});
</script>
