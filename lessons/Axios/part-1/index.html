Mastering API Requests: Fetch, Dev Tools, Axios

This module teaches you how to fetch data from APIs using Native Fetch, debug with Chrome Network Dev Tool, and simplify requests with Axios. We’ll use the Star Wars API (SWAPI) and V School Todo API.

1. Introduction to Native Fetch

The fetch API is a built-in JavaScript tool for making HTTP requests, like getting data from APIs. It’s modern, promise-based, and replaces older methods like XMLHttpRequest.

  • Why Use Fetch? It’s simple, supports promises, and works in browsers without extra libraries.
  • How It Works: Fetch sends a request to a URL and returns a promise that resolves with the response.

Example: Fetching Luke Skywalker

Fetch data from SWAPI:


fetch("https://swapi.co/api/people/1")
    .then(response => response.json())
    .then(data => console.log(data.name)) // "Luke Skywalker"
    .catch(error => console.log("Error:", error));
