4. APIs and JSON

APIs (Application Programming Interfaces) let apps talk to servers. JSON (JavaScript Object Notation) is the modern format for API data, similar to JavaScript objects.

  • APIs: Provide data (e.g., SWAPI for Star Wars info).
  • JSON: Structured like {key: value}, parsed into JavaScript objects.

Example: Parsing JSON with Fetch

Fetch and parse SWAPI JSON:


fetch("https://swapi.co/api/planets/1")
    .then(res => res.json())
    .then(data => console.log(data)) // {name: "Tatooine", ...}
    .catch(err => console.log("Error:", err));

Mini Exercise (5 min): Visit https://swapi.co/api/people/1 in your browser. Identify one JSON key-value pair (e.g., "name": "Luke Skywalker"). Use fetch to log that pair in the console.
