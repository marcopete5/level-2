10. Axios PUT Requests

Axios’s put method updates existing resources on a server, like editing a todo in the V School API.

  • Syntax: axios.put(url, data) targets a specific resource by ID.
  • Use Case: Update fields (e.g., title, description).

Example: Updating a Todo

Update a todo’s title:


const updates = {
    title: "Updated Todo",
    description: "New description"
};
axios.put("https://api.vschool.io/yourname/todo/your-todo-id", updates)
    .then(response => console.log(response.data))
    .catch(error => console.log("Error:", error));
