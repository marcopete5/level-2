2. Understanding Fetch Promises

Fetch returns a promise, an object that represents a future result (data or error). Promises have states: Pending (waiting), Resolved (success), or Rejected (failed).

  • .then(): Handles Resolved data (e.g., JSON).
  • .catch(): Handles Rejected errors (e.g., network failure).

Example: Handling Fetch Promises

Fetch and process SWAPI data:


fetch("https://swapi.co/api/people/2")
    .then(res => res.json())
    .then(data => console.log(data.name)) // "C-3PO"
    .catch(err => console.log("Error:", err));
