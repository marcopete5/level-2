5. Updating with PUT Requests

PUT requests update existing data in an API’s database. Use the V School API to edit a specific todo item by its _id.

  • Setup: Select PUT, use https://api.vschool.io/[your-name]/todo/[_id], set Body to raw and JSON (application/json).
  • Fields: Update title, description, price, imgUrl, or completed.

Example: PUT Request

Update a todo’s completed status:


{
    "completed": true
}

Mini Exercise: In Postman, send a GET to https://api.vschool.io/[your-name]/todo/ to find a todo’s _id. Then, send a PUT request to update its title. Verify the updated item in the response.
