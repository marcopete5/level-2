Mastering Postman: Your API Testing Tool

4. Making POST Requests

POST requests save new data to an API’s database, making Postman invaluable. You’ll use the V School API to create todo items.

  • Setup: Select POST, use https://api.vschool.io/[your-name]/todo/, set Body to raw and JSON (application/json).
  • JSON Format: Include a title (required), optional description, price, imgUrl, completed.

Example: POST Request

For user “fred”:


{
    "title": "Learn Postman",
    "description": "Practice API requests",
    "price": 0,
    "completed": false
}

Mini Exercise: In Postman, send a POST request to https://api.vschool.io/[your-name]/todo/ with a JSON todo (e.g., {"title": "Test Postman"}). Check the response for your new item.
