6. Deleting with DELETE Requests

DELETE requests remove data from an API’s database. Use the V School API to delete a specific todo item by its _id.

  • Setup: Select DELETE, use https://api.vschool.io/[your-name]/todo/[_id], no body needed.
  • Response: Expect {"msg": "Successfully deleted record"}.

Example: DELETE Request

Delete a todo item:
