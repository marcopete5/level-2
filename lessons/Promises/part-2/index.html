Promises: Synchronous vs. Asynchronous Programming

2. Synchronous Code Examples

Synchronous code, or blocking code, executes one statement at a time, waiting for each to complete before moving to the next.

  • Characteristics: Predictable order, but can slow down apps if tasks take long.
  • Examples: Variable assignments, simple functions, recursive calculations.

Example: Synchronous Code

Each line waits for the previous one:


var firstName = "Joe";
function addExcitement(str) {
    return str + "!!1!";
}
console.log(addExcitement("Hi")); // "Hi!!1!"
function factorial(n) {
    if (n === 1) return 1;
    return n * factorial(n - 1);
}
console.log(factorial(5)); // 120
