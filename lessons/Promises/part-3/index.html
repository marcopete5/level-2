Promises: Synchronous vs. Asynchronous Programming

3. Asynchronous Code Examples

Asynchronous code, or non-blocking code, starts tasks without waiting, allowing other code to run, ideal for slow operations like API calls or user input.

  • Characteristics: Non-predictable order, improves app responsiveness.
  • Examples: Timers, event listeners, API requests.

Example: Asynchronous Code

Non-blocking execution:


setTimeout(() => console.log("Inside setTimeout"), 500);
console.log("Outside setTimeout"); // Runs first
document.getElementById("thing").addEventListener("click", () => console.log("Clicked!"));
fetch("https://swapi.dev/api/people/1/")
    .then(res => res.json())
    .then(data => console.log(data.name)) // Runs second
console.log("Outside fetch"); // Runs first
