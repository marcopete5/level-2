3. XHR open() Method

The open() method configures an XHR request before sending it.

  • Parameters:
    • method: HTTP method (e.g., "GET" to retrieve, "POST" to submit).
    • url: The API endpoint (e.g., https://swapi.dev/api/people/1/).
    • async: Boolean for asynchronous requests (usually true).
  • Purpose: Sets up the requestâ€™s type, destination, and behavior.

Example: Using open()

Configure a GET request:


const xhr = new XMLHttpRequest();
xhr.open("GET", "https://swapi.dev/api/people/1/", true);
