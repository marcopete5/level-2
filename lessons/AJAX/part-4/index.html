4. XHR send() Method

The send() method initiates the XHR request, sending it to the server.

  • Role: Triggers the request configured by open().
  • Usage: For GET requests, send() takes no arguments. For POST, it can include data (e.g., JSON).

Example: Using send()

Send a GET request:


const xhr = new XMLHttpRequest();
xhr.open("GET", "https://swapi.dev/api/people/1/", true);
xhr.send();
