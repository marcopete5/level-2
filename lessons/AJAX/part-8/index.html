Mastering AJAX: Dynamic Web Updates

8. Using AJAX Data and Conclusion

Once you receive and parse AJAX data, you can update the web page dynamically. Modern libraries simplify this process.

  • Using Data: Update the DOM with parsed JSON (e.g., display a character’s name).
  • Modern Tools: Libraries like Axios and jQuery simplify AJAX, but understanding XHR helps debug issues.

Example: Displaying Data

Fetch and display a Star Wars character’s name:


const xhr = new XMLHttpRequest();
xhr.onreadystatechange = function () {
    if (xhr.readyState === 4 && xhr.status === 200) {
        const data = JSON.parse(xhr.responseText);
        document.getElementById("demo").textContent = data.name;
    }
};
xhr.open("GET", "https://swapi.dev/api/people/1/", true);
xhr.send();
