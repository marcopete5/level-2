6. XHR readyState and status

The readyState and status properties track an XHR request’s progress and outcome.

  • readyState: Indicates the request’s stage (0: unsent, 1: opened, 2: headers received, 3: loading, 4: done).
  • status: HTTP status code (e.g., 200 for success, 404 for not found).

Example: Checking readyState and status

Verify a successful response:


xhr.onreadystatechange = function () {
    if (xhr.readyState === 4 && xhr.status === 200) {
        console.log("Request successful!");
    }
};
