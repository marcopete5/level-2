Mastering AJAX: Dynamic Web Updates

This 60-minute module introduces AJAX, a technology that enables web pages to update data without reloading. You’ll learn how to use the XMLHttpRequest (XHR) object to make asynchronous requests.

1. Introducing AJAX

AJAX (Asynchronous JavaScript and XML) allows browsers to communicate with servers without refreshing the page, using the XMLHttpRequest (XHR) object.

  • Why It Matters: Before AJAX, updating content (e.g., new emails in Gmail) required reloading the entire page, resending all HTML, CSS, and JavaScript—slow and inefficient.
  • Impact: By 2003, browsers adopted XHR, enabling apps like Google Maps and Gmail to fetch new data (e.g., map tiles, emails) seamlessly.

Example: AJAX in Action

AJAX fetches JSON data from an API without reloading:


const xhr = new XMLHttpRequest();
xhr.open("GET", "https://swapi.dev/api/people/1/", true);
xhr.send();
