2. Creating an XHR Object

The XMLHttpRequest (XHR) object is the core of AJAX, enabling asynchronous requests to servers.

  • How It Works: Use the XMLHttpRequest constructor to create an XHR object, often called xhr for simplicity.
  • Purpose: The object handles communication with the server, fetching or sending data without reloading the page.

Example: Creating XHR

Create a new XHR object:


const xhr = new XMLHttpRequest();
