JSON's Deli - Assignment

In this assignment, you’ll apply your JSON skills to create a menu for JSON's Deli, a new sandwich shop in Silicon Valley catering to programmers. You’ll write the menu in JSON format to delight your tech-savvy customers.

You’re opening JSON's Deli, and you need a menu in JSON format to display all your items. Follow these requirements to create a valid JSON file.

Requirements

  • File Format: Save your JSON as menu.json. The JSON must be an object ({}) representing the entire menu.
  • Menu Structure: The JSON object should have five properties, each an array, representing:
    • appetizers
    • coldSandwiches
    • hotSandwiches
    • desserts
    • drinks
  • Item Details: Each array item is an object with three properties:
    • name: The item’s name (string).
    • ingredients: An array of strings listing the ingredients.
    • description: A string describing the item.
  • Minimum Items: Include at least two items (objects) in each submenu array (appetizers, cold sandwiches, hot sandwiches, desserts, drinks).

Tips

  • Validate Your JSON: Periodically paste your JSON into JSONLint to ensure correct syntax (e.g., double quotes on keys, no trailing commas).
  • Be Creative: Name your items with programmer-friendly themes (e.g., “Byte-Sized Brownie”) to appeal to your Silicon Valley customers.

Assignment Task: Create menu.json with at least two items per submenu, following the structure above. Validate it using JSONLint.
