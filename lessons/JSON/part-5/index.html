Understanding JSON: Storing and Exchanging Data

5. JSON in APIs

JSON is the standard format for data exchange in web APIs. You’ll use it to fetch data from external services or send data to servers.

  • How It Works: APIs return JSON data, which your app can parse and use.
  • Example Use: Fetching user data from an API like Instagram.

Example: Fetching JSON

Use fetch to get JSON from an API.


fetch('https://jsonplaceholder.typicode.com/users/1')
    .then(response => response.json())
    .then(data => console.log(data));

Response (JSON):


{
    "id": 1,
    "name": "Leanne Graham",
    "username": "Bret",
    "email": "Sincere@april.biz"
}
