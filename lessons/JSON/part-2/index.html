Understanding JSON: Storing and Exchanging Data

2. JSON vs. JavaScript Objects

JSON is very similar to JavaScript objects but has strict rules that make it a data format, not a programming construct.

  • Similarities: Both use key-value pairs; values can be strings, numbers, arrays, or nested objects.
  • Differences:
    • JSON keys require double quotes ("key"), unlike JavaScript objects.
    • JSON values cannot be functions/methods, only data (strings, numbers, arrays, objects).
    • JSON uses double quotes for strings, not single quotes.

Example: JSON vs. JavaScript


{
    "name": "Jane",
    "age": 25
}

JavaScript Object:


const person = {
    name: "Jane",
    age: 25,
    greet: function() { console.log("Hi!"); }
};
