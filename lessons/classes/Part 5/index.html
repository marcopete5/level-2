5. Random Events and Timers

In games, random events and timed actions are common. Use a random range function and setInterval to simulate game events.

Example: Random Events


function getRandomRange(min, max) {
    return Math.floor(Math.random() * (max - min + 1)) + min;
}

const player = new Player("Luigi", 0, "Small", false);
const gameInterval = setInterval(() => {
    const action = getRandomRange(0, 2);
    if (action === 0) player.gotHit();
    else if (action === 1) player.gotPowerup();
    else player.addCoin();
    player.print();
    if (player.status === "Dead") clearInterval(gameInterval);
}, 1000);

Key Notes:

  • getRandomRange(0, 2) returns 0, 1, or 2 randomly.
  • setInterval runs code every X milliseconds until stopped with clearInterval.
