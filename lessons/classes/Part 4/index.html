4. Managing State with Methods

Classes can include methods to update object state, such as changing properties based on conditions. This is useful for game logic, like updating a player's status.

Example: State-Changing Methods


class Player {
    constructor(name, totalCoins, status, hasStar) {
        this.name = name;
        this.totalCoins = totalCoins;
        this.status = status;
        this.hasStar = hasStar;
    }
    gotHit() {
        if (this.status === "Powered Up") this.status = "Big";
        else if (this.status === "Big") this.status = "Small";
        else if (this.status === "Small") this.status = "Dead";
    }
    gotPowerup() {
        if (this.status === "Small") this.status = "Big";
        else if (this.status === "Big") this.status = "Powered Up";
        else if (this.status === "Powered Up") this.hasStar = true;
    }
}
