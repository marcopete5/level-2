The Princess is in Another Castle

Assignment

Apply what you've learned in this comprehensive exercise.

Requirements

  • Create a Player class with:
    • Properties: name (String), totalCoins (Number), status (String: "Powered Up", "Big", "Small", "Dead"), hasStar (Boolean)
    • Methods:
      • setName(namePicked): Sets name to "Mario" or "Luigi" based on namePicked
      • gotHit(): Updates status (Powered Up → Big → Small → Dead)
      • gotPowerup(): Updates status (Small → Big → Powered Up, or sets hasStar if Powered Up)
      • addCoin(): Increments totalCoins
      • print(): Logs name, coins, status, and star in a formatted way
  • Instantiate a Player object.
  • Create a getRandomRange(min, max) function that returns a random integer (0, 1, or 2).
  • Use the random function to call gotHit() (0), gotPowerup() (1), or addCoin() (2), then call print().
  • Run the random actions in a setInterval that stops when the player is "Dead".
