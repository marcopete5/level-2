JavaScript Classes: A Modern Approach

3. Inheritance with extends and super

Inheritance allows a class to inherit properties and methods from another class using extends. The super keyword calls the parent class's constructor to initialize inherited properties.

Example: Extending a Class


class Vehicle {
    constructor({ make, model }) {
        this.make = make;
        this.model = model;
    }
    drive() {
        console.log("vroom");
    }
}

class Motorcycle extends Vehicle {
    constructor({ make, model, hasSideCar }) {
        super({ make, model }); // Call parent constructor
        this.hasSideCar = hasSideCar;
    }
}

const bike = new Motorcycle({ make: "Honda", model: "Shadow", hasSideCar: false });
console.log(bike.make); // Honda
bike.drive(); // vroom

Key Notes:

  • Use super before accessing this in the child constructor.
  • Destructuring in the constructor makes passing objects cleaner.
