Understanding HTTP: The Web’s Communication Protocol

4. APIs and the Request-Response Cycle

An API (Application Programming Interface) is code on a server that handles specific requests, fetching or saving data. The request-response cycle describes the process: a client sends an HTTP request, the server’s API processes it, and the server sends a response (often as JSON).

Example: Fetching Data


fetch('https://api.example.com/products')
    .then(response => response.json())
    .then(data => console.log(data));

Response (JSON):


[
    { "id": 1, "name": "Boots", "price": 99.99 },
    { "id": 2, "name": "Sneakers", "price": 59.99 }
]
