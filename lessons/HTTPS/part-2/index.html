Understanding HTTP: The Web’s Communication Protocol

2. IP Addresses and DNS

An IP Address is a unique identifier for each device on a network, like a mailing address for your computer. It ensures servers send data to the correct client.

  • Format: Four numbers (0–255) separated by periods, e.g., 192.168.1.1.
  • DNS (Domain Name System): Translates human-readable URLs (e.g., google.com) into IP addresses.

Example: Resolving a URL

When you visit amazon.com, DNS resolves it to an IP address like 54.239.28.85. Your browser then sends an HTTP request to that IP. 


GET / HTTP/1.1
Host: amazon.com
