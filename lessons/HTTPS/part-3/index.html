Understanding HTTP: The Web’s Communication Protocol

3. Servers

Servers are computers that store and deliver web data. They listen for HTTP requests and send responses.

  • Remote Servers: Hosted elsewhere (e.g., Amazon’s servers).
  • Local Servers: Run on your computer for development (e.g., using Node.js).

Example: Basic Node.js Server


const http = require('http');
const server = http.createServer((req, res) => {
    res.writeHead(200, { 'Content-Type': 'text/plain' });
    res.end('Hello, World!');
});
server.listen(3000, () => console.log('Server running on port 3000'));

Run this code, then visit http://localhost:3000 to see “Hello, World!”.
