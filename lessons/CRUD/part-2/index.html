Mastering CRUD: Core Data Operations in Web Development

2. Create Operation

The Create operation adds new records or resources to a database, typically using the HTTP POST method in REST APIs.

  • Purpose: Store new data, like a new user or post.
  • Response: Often returns a 201 Created status with the new resource’s details.

Example: Creating a User

Send a POST request to add a user to a database.


fetch('https://api.example.com/users', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ name: 'Alice', email: 'alice@example.com' })
})
    .then(response => response.json())
    .then(data => console.log(data));

Response (JSON):


{ "id": 1, "name": "Alice", "email": "alice@example.com" }

Mini Exercise: In your browser’s console, use fetch to send a POST request to https://jsonplaceholder.typicode.com/users with a sample user object. Log the response.
