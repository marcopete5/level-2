Mastering CRUD: Core Data Operations in Web Development

5. Delete Operation

The Delete operation removes records from a database, using the HTTP DELETE method.

  • Purpose: Remove data, like deleting a user account.
  • Response: Often returns a 204 No Content status to confirm deletion.

Example: Deleting a User

Send a DELETE request to remove a user.


fetch('https://api.example.com/users/1', {
    method: 'DELETE'
})
    .then(response => console.log(response.status)); // 204
