Mastering CRUD: Core Data Operations in Web Development

4. Update Operation

The Update operation modifies existing records in a database, using the HTTP PUT method (or sometimes PATCH for partial updates).

  • Purpose: Change data, like updating a user’s email.
  • Response: Usually returns a 200 OK or 204 No Content status.

Example: Updating a User

Send a PUT request to update a user’s email.


fetch('https://api.example.com/users/1', {
    method: 'PUT',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ email: 'alice@newemail.com' })
})
    .then(response => response.json())
    .then(data => console.log(data));

Response (JSON):


{ "id": 1, "name": "Alice", "email": "alice@newemail.com" }
