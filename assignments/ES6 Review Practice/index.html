1. Property Shorthand and Method Shorthand

Create an object representing a book using property and method shorthand. The book should have title, author, and a method getDescription() that returns a description.

Expected Output: { title: "JavaScript Guide", author: "Jane Doe", getDescription: [Function] } and "JavaScript Guide by Jane Doe" when calling getDescription(). 


const title = "JavaScript Guide";
const author = "Jane Doe";
const book = {
    // TODO: Use property shorthand for title and author
    // TODO: Add a getDescription method using method shorthand
};
console.log(book);
console.log(book.getDescription());
See Solution 

const title = "JavaScript Guide";
const author = "Jane Doe";
const book = {
    title,
    author,
    getDescription() {
        return `${this.title} by ${this.author}`;
    }
};
console.log(book);
console.log(book.getDescription());

See Solution

5. Rest Operator

Write a function that takes any number of arguments and returns their average. Use the rest operator to collect arguments into an array.

Expected Output: 3 (for inputs 1, 2, 3, 4, 5) 


function calculateAverage(/* TODO: Use rest operator */) {
    // TODO: Calculate and return the average of the numbers
}
console.log(calculateAverage(1, 2, 3, 4, 5));
See Solution 

function calculateAverage(...numbers) {
    return numbers.reduce((sum, num) => sum + num, 0) / numbers.length;
}
console.log(calculateAverage(1, 2, 3, 4, 5));

8. Let and Const

Expected Output: 1, 2, 3 (without variable leakage)

    // TODO: Use let/const in a for loop to log numbers 1 to 3

See Solution

10. Object Destructuring
