Axios Todo Project

Capstone Project

Build it, own it! Create a dynamic Todo List using V School Todo API.

You’ll build a frontend site that pulls todos from the V School Todo API, displays them, and supports CRUD operations (Create, Read, Update, Delete). Changes persist in the database, so refreshing the page shows the current todo list.

Setup

Use Axios via CDN to make API requests. Work through each part in order, collaborating with peers between steps to discuss code and explore new approaches.

Workshops to Review

  • Project Set-Up / Vanilla JS, Front-end, Full-Stack
  • Functions and Parameters
  • Minimal Viable Product (MVP) / CSS Tips and Tricks

General Requirements

Instructors will check for functionality, responsiveness, and similarity to the demo app. Expect a discussion on challenges and helpful resources. Spend 1–3 days on JavaScript and 1–2 days on CSS.

Total Story Points: 8
