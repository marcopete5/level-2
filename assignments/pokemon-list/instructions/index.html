Pokémon API

Mastery Task

This Mastery Task challenges you to use Axios in vanilla JavaScript to fetch and display Pokémon data from https://api.vschool.io/pokemon. You’ll practice HTTP requests, JSON parsing, and DOM manipulation to show a list of Pokémon names in the browser.

Complete the task in two steps: fetch the data, then display it. Use only vanilla JavaScript (no frameworks).

Step One: Get the Data

Write a function that uses an Axios GET request to fetch Pokémon data from https://api.vschool.io/pokemon. The response data is structured like this: 


{
    "meta": {
        "limit": 1,
        "next": null,
        "offset": 0,
        "previous": null,
        "total_count": 1
    },
    "objects": [
        {
            "created": "2013-11-09T15:14:48.957604",
            "modified": "2013-11-09T15:14:48.957565",
            "name": "national",
            "pokemon": [
                {
                    "name": "rattata",
                    "resource_uri": "api/v1/pokemon/19/"
                },
                {
                    "name": "charmander",
                    "resource_uri": "api/v1/pokemon/4/"
                },
                ...etc
            ]
        }
    ]
}

Tip: You can’t use Node.js without extra setup, so create an HTML file linking your JavaScript and run it in a browser. Include Axios via CDN.

Step Two: Display the Data

Display each Pokémon’s name on a separate line in the browser using DOM manipulation. Use a for loop to iterate through the Pokémon array and create elements (e.g., p or h1) for each name.

  • Note: The response data is nested. Access the Pokémon array at response.data.objects[0].pokemon.
  • Example: Create a p element for each name (e.g., “rattata”, “charmander”) and append it to a container div.

Hints

  • Use the Network tab (F12 on Windows, Command + Option + I on Mac) to inspect the data returned from the HTTP request. Check the “Response” tab to understand the JSON structure.
  • Log response.data to the console to explore the nested objects before coding the loop.
  • Start with a simple HTML file and a div to hold the Pokémon names.

Starter Code

Use this HTML to begin, and create index.js for your logic: 


<!-- index.html -->
<!DOCTYPE html>
<html>
    <head>
        <title>Pokémon List</title>
    </head>
    <body>
        <!-- Container for Pokémon names -->
        <div id="pokemon-list"></div>
        <!-- Include Axios for API requests -->
        <script src="https://cdnjs.cloudflare.com/ajax/libs/axios/0.19.0/axios.js"></script>
        <script src="index.js"></script>
    </body>
</html>
