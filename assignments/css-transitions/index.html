CSS Transitions Practical Demo

1. Basic Button Transition

Hover over the button to see a smooth color transition:

CSS used:


.basic-button {
    background-color: blue;
    transition: background-color 0.3s ease;
}

.basic-button:hover {
    background-color: darkblue;
}

2. Multiple Property Transitions

Hover over the card to see multiple properties transition with different timings:

Hover Me

CSS used:


.card {
    width: 200px;
    height: 300px;
    opacity: 0.8;
    transition: width 0.5s ease, 
                height 0.8s ease-in,
                opacity 0.3s linear;
}

.card:hover {
    width: 220px;
    height: 320px;
    opacity: 1;
}

3. Transform with Transitions

Hover over the card to see a transform effect with transition:

Hover to Scale

CSS used:


.grow-card {
    transform: scale(1);
    transition: transform 0.3s ease-out;
}

.grow-card:hover {
    transform: scale(1.1);
}

5. Menu Underline Effect

Home About Services Contact

CSS used:


.nav-link {
    position: relative;
}

.nav-link::after {
    content: '';
    position: absolute;
    bottom: 0;
    left: 0;
    width: 0;
    height: 2px;
    background-color: #ff6b6b;
    transition: width 0.3s ease;
}

.nav-link:hover::after {
    width: 100%;
}
