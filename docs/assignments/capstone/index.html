Axios Todo List

Level 2 Capstone Project
  • Build a Todo List: Create a frontend site using the V School Todo API to manage todos with CRUD operations.
  • Core Functionality:
    • GET: Display todos on load with strikethrough for completed items and images (2 points).
    • POST: Add todos with title, price, description, and optional image, persisting via API (2 points, extra credit for no refresh).
    • PUT: Toggle completion with checkboxes, updating API (1 point).
    • DELETE: Remove todos with a trash icon (1 point).
  • Styling & Responsiveness: Use Flexbox/Grid and media queries for a responsive design matching Any.do’s look, tested at phone, tablet, and desktop (2 points).
  • Development: Use Axios via CDN, spend 1–3 days on JavaScript, 1–2 days on CSS, and collaborate with peers (8 story points total).
  • Tips: Test API with Postman and review workshops on project setup, functions, and CSS.
