Pokémon List

Assignment

Instructions

The following exercise should be completed in vanilla JavaScript.

Use https://api.vschool.io/pokemon as your URL for your Axios request to get the big list of Pokémon.

It will be easiest to do this in steps:

  • Step one - get the data
  • Step two - display the data

Step One

Write a function that runs an axios get request, the response data will be structured something like this: 


{
    "meta": {
        "limit": 1,
        "next": null,
        "offset": 0,
        "previous": null,
        "total_count": 1
    },
    "objects": [
        {
            "created": "2013-11-09T15:14:48.957604",
            "modified": "2013-11-09T15:14:48.957565",
            "name": "national",
            "pokemon": [
                {
                    "name": "rattata",
                    "resource_uri": "api/v1/pokemon/19/"
                },
                {
                    "name": "charmander",
                    "resource_uri": "api/v1/pokemon/4/"
                },
                ...etc
            ]
        }
    ]
}

Tip: You won't be able to use node for this without extra work so you'll need to use an HTML file that links to your JavaScript and run it in the browser.

Step Two

Make each Pokémon's name show up on a separate line in the browser.

You will be using a for loop to iterate through each pokemon object, and some DOM manipulation to add nodes for each Pokémon.

  • The data that comes from the URL endpoint above is complex. Pay attention to the structure of the response data to access the array of pokemon.
Hints
  • Use the network tab view data you’re getting back from the HTTP request

Submitting Assignments

Please remember to regularly submit your assignments to GitHub!

Using Git Command Line:

  1. Open Terminal and navigate to your project folder.
  2. Add Changes: git add <relative/path/to/assignment/folder>
  3. Commit Changes: git commit -m "submitting assignment <name>"
  4. Push to GitHub: git push origin main

Using VSCode's Source Control:

  1. Open VSCode and click on the Source Control tab.
  2. Stage Changes: Click the "+" icon next to each file.
  3. Commit Changes: Enter a commit message and click Commit.
  4. Push to GitHub: Click the Sync Changes button.
