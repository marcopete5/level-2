Using Axios for DELETE Requests

Introduction to DELETE Requests

DELETE requests are used to remove resources from a server. In this guide, we'll explore how to use Axios to make DELETE requests in JavaScript.

Understanding the Code

Let's break down the provided code step by step:


const button = document.getElementById('delete-button')

button.addEventListener("click", function(){
    axios.delete("<https://api.vschool.io/scrimbalessons/todo/5d8bd511ee91575e6d49e06e>")
        .then(response => console.log(response.data))
        .catch(error => console.log(error))
})

Capturing Button Click

  • We select the delete button from the document using document.getElementById('delete-button').
  • We add an event listener to the button for the "click" event.

Sending DELETE Request with Axios

  • When the button is clicked, Axios sends a DELETE request to the specified API endpoint (https://api.vschool.io/scrimbalessons/todo/5d8bd511ee91575e6d49e06e).
  • The ID 5d8bd511ee91575e6d49e06e represents the specific resource (todo in this case) we want to delete.
  • If the request is successful, we log the response data to the console.
  • If an error occurs, we log the error to the console.
